# Nur Cosmetic Shop App

`Nur` is now a browser-based cosmetic shop app built with plain HTML, CSS, and JavaScript.

## Included features

- Login and registration flow
- Home, Shop, Favorites, Cart, and Profile pages
- Product detail screen
- Category filters and search
- Favorites toggle
- Add to cart, quantity updates, and checkout flow
- Editable profile and logout

## Run locally

Option 1:
Open `index.html` directly in your browser.

Option 2:
If you want a local server, run:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Notes

- The app uses local browser state only, so it works without a backend.
- Authentication and shopping actions are demo-functional and run entirely on the client side.
